# Frontend Components

Key UI components that power the frontend.

Source: `frontend/src/components/`

## Meta Form Components

### SchemaField (`meta/SchemaField.tsx`)
Recursive field renderer that takes a form schema (generated by [[meta-form-schema-engine]]) and renders the appropriate input for each field type:
- `string` → text input
- `number` → number input
- `enum` → select/dropdown with valid values
- `boolean` → toggle/checkbox
- `object` → nested field group (recursive)
- `array` → list editor

### MetaForm (`meta/MetaForm.tsx`)
Stateful form wrapper that manages form state, handles field changes, and calls the backend for schema updates when dependent fields change (e.g. changing objective updates valid optimization goals).

## Wide Creation Components (`wide-create/`)

### StructureWizard
Step-by-step wizard for defining a wide creation template:
- Set campaign count and objective
- Configure adset count per campaign
- Configure ad count per adset
- Set field defaults and naming patterns

### TreeConfigurator
Visual tree view showing the generated campaign structure. Each node can be expanded and configured individually. Changes are saved to the Zustand store.

## Layout Components (`layout/`)
App-level layout: sidebar navigation, top bar, account selector.

## Dashboard Components (`dashboard/`)
Dashboard widgets showing overview stats and quick actions.

## FacebookSDK (`FacebookSDK.tsx`)
Initializes the Facebook JavaScript SDK v21.0. Used on the login page for OAuth authentication.
